justnewbee.github.io
====================

# how

1. install jekyll
	* require ruby > 2.1
	* require bundler - `gem install bundler`

## run

```
bundle install
bundle exec jekyll serve
```

## upgrade

```
gem update bundler
gem update jekyll
```
